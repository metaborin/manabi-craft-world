# まなびクラフトワールド / manabi-craft-world

小学生が、探索・クエスト・建築を通して学べる、オリジナルの3Dボクセル学習ゲームです。

プレイヤーは学年を選び、3Dワールドを歩きながら先生NPCに話しかけ、国語・算数・理科・社会・英語・生活の学習クエストに挑戦します。クエストで集めたコイン、経験値、ブロックを使って、建築エリアで自分だけのワールドづくりを楽しめます。

## 主な機能

- 小学1年生から6年生までの学年切り替え
- 国語・算数・理科・社会・英語・生活の学習クエスト
- 先生NPCから始まる3問1セットのクエスト
- ヒント、追加ヒント、正解ハイライトによる学習サポート
- 3Dボクセル風ワールドの探索
- セーブデータ2つ
- コイン、経験値、レベル、バッジ
- クエスト報酬で集めたブロックを使う建築エリア
- ショップでのブロックやペットのたまご交換
- PC操作とタッチ操作への対応
- PWA化を想定したmanifest同梱

## 技術構成

- Vite
- React
- TypeScript
- Three.js / React Three Fiber
- Zustand
- localStorage

## 起動方法

```bash
npm install
npm run dev
```

起動後、ブラウザで以下を開きます。

```text
http://localhost:5173
```

## ビルド方法

```bash
npm run build
```

ビルド成果物は `dist` に出力されます。`dist` は通常Git管理しません。

## あそびかた

- タイトル画面でセーブスロットを選ぶ
- なまえ、すきな色、学年を選んでワールドへ入る
- PCではWASD/矢印キーで移動、スペースでジャンプ、E/Enterで話す、ドラッグでカメラ回転
- タブレットでは左下の移動パッド、右下のジャンプボタン、画面ドラッグでカメラ操作
- 先生NPCに話しかけると学習クエストが始まる
- クエスト報酬を集めて、建築エリアやショップで使う

## 問題の追加方法

`src/data/questions.ts` の `QUESTIONS` 配列に問題オブジェクトを追加します。ファイル冒頭のコメントに記入例があります。

- `grade`: 1から6
- `subject`: `kokugo` / `sansu` / `rika` / `shakai` / `eigo` / `seikatsu`
- `answer`: 正解の選択肢番号。0から数えます
- 問題文、選択肢、解説には `{漢字|かんじ}` の形でふりがなを入れられます
- `visual` に絵文字を入れると、問題の下に大きく表示されます

## 主なファイル構成

| 場所 | 内容 |
| --- | --- |
| `src/data/questions.ts` | 問題データ |
| `src/data/grades.ts` | 学年・教科の定義 |
| `src/data/rewards.ts` | ブロック・バッジ・ペット・ボーナス |
| `src/data/world.ts` | ワールドのNPC・看板の配置 |
| `src/types/game.ts` | 型定義 |
| `src/store/gameStore.ts` | ゲーム状態・クエスト進行ロジック |
| `src/store/saveSystem.ts` | localStorageセーブ |
| `src/game/` | 3Dワールド描画・入力 |
| `src/screens/` | タイトル、学年選択、クエスト、建築、ショップなどの画面 |
