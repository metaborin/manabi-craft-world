# まなびクラフトワールド / manabi-craft-world

小学生が、探索・クエスト・建築を通して学べる、オリジナルの3Dボクセル学習ゲームです。

プレイヤーは学年を選び、3Dワールドを歩きながら先生NPCに話しかけ、国語・算数・理科・社会・英語・生活の学習クエストに挑戦します。クエストで集めたコイン、経験値、ブロックを使って、建築エリアで自分だけのワールドづくりを楽しめます。

## 主な機能

- 小学1年生から6年生までの学年切り替え
- 国語・算数・理科・社会・英語・生活の学習クエスト
- 先生NPCから始まる3問1セットのクエスト
- ヒント、追加ヒント、正解ハイライトによる学習サポート
- 3Dボクセル風ワールドの探索
- セーブデータ2つ
- コイン、経験値、レベル、バッジ
- クエスト報酬で集めたブロックを使う建築エリア
- ショップでのブロックやペットのたまご交換
- PC操作とタッチ操作への対応
- はじめてのプレイをあんないするチュートリアル（6ステップ）
- 「つぎにすること」を常に表示するガイド
- せってい画面（もじのおおきさ・ふりがな・画面内ボタン・こうかおん）
- PWA化を想定したmanifest同梱

## あそびかた

- タイトル画面でセーブスロットを選ぶ
- なまえ、すきな色、学年を選んでワールドへ入る
- PCではWASD/矢印キーで移動、スペースでジャンプ、E/Enterで話す、ドラッグでカメラ回転
- タブレットでは左下の移動パッド、右下のジャンプボタン、画面ドラッグでカメラ操作
- 先生NPCに話しかけると学習クエストが始まる
- クエスト報酬を集めて、建築エリアやショップで使う

## 問題の追加方法

`src/data/questions.ts` の `QUESTIONS` 配列に問題オブジェクトを追加します。ファイル冒頭のコメントに記入例があります。

先生・保護者が問題を追加するときの手順:

1. `QUESTIONS` 配列に `{ ... }` を1つ追加する
2. `id`: 他とかぶらない文字列（例 `g1-sansu-tashizan-99`）
3. `grade`: 学年 1〜6
4. `subject`: `kokugo` / `sansu` / `rika` / `shakai` / `eigo` / `seikatsu`
5. `unit`: 単元名（クエスト画面のヘッダーに表示されます）
6. `difficulty`: むずかしさ 1〜5
7. `question` / `choices` / `answer`(0はじまりの正解番号) を入れる
8. `hint`（1回目）と `hint2`（2回目・より具体的）を書く
9. `explanation`: 正解後の短い解説
10. `reward`: `{ coins, xp, blocks? }`（例 `{ coins: 6, xp: 12 }`）

### 漢字とふりがな（読み支援）

問題文・選択肢・ヒント・解説・単元名すべてで、`{漢字|よみ}` の形にすると読み支援になります。

- ふりがな **ON**（初期設定）… `{山|やま}` → 山（やま）とルビ付きで表示
- ふりがな **OFF**（設定で切替）… `{山|やま}` → 山 と漢字だけ表示

**学年で習う漢字だけ `{漢字|よみ}` を使い、未習の漢字はひらがなで書く**のが方針です（小1=1年生漢字、小2=1〜2年生、小3=1〜3年生）。これで、読める子には漢字、まだの子にはふりがな、と両対応できます。

`visual` に絵文字を入れると、問題の下に大きく表示されます。

## 先生・保護者の方へ

- 本作は、小学生が探索・建築・収集を楽しみながら、国語・算数・理科・社会・英語・生活の問題に取り組める学習ゲームです
- 「まちがえても責めない」設計です。不正解にはやさしいヒントが出て、最後は正解を一緒に確認できます
- **個人情報は収集・送信しません**。ニックネームやセーブデータは、端末のブラウザ内（localStorage）にのみ保存されます
- 外部サーバーとの通信・ログイン・課金はありません
- 学年は1〜6年をいつでも切り替えられます。問題データは `src/data/questions.ts` に追記するだけで増やせます
- ゲーム内の「❓あそびかた」画面にも、児童向けの説明と保護者向けの案内があります
